CNJ Digital — Email Signatures

Each person has a full signature (new emails) and a short reply version.

How to set up your Gmail signature

1 Open Gmail Settings

Click the ⚙️ gear icon in the top-right corner of Gmail, then click "See all settings".

Gmail toolbar → ⚙️
Quick settings
See all settings
2 Create two signatures

In the General tab, scroll to Signature. Click "+ Create new" twice and name them exactly:

✉️
CNJ full
Used for new emails you send
↩️
CNJ reply
Used when replying or forwarding
3 Paste each signature

For each signature: select its name on the left → click inside the editor → paste with Cmd+V (Mac) or Ctrl+V (Windows).

✉️ CNJ full → paste the full signature (black button below)
↩️ CNJ reply → paste the short reply version (white button below)
4 Set defaults & save

Scroll to Signature defaults and set:

✉️ FOR NEW EMAILS USE
CNJ full ▾
The full branded signature
↩️ ON REPLY/FORWARD USE
CNJ reply ▾
The short text-only version
Save Changes
